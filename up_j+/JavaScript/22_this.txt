Ключевое слово this указывает на тот объект, в контексте которого оно было вызвано.

Некоторые нюансы:
  - Контекстом вызова this обычной функции является глобальный объект Window.

  - В режиме use strict контекстом this обычной функции будет undefined.

  - Контекстом вызова метода объекта является сам объект.

  - Стрелочная функция не имеет своего контекста this, контекстом для неё является
    контекст вызова родителя.

bind()
  - создаёт новую функцию, которая при вызове устанавливает в качестве контекста
    выполнения this предоставленное значение;
  
call()
  - вызывает функцию с указанным значением this и индивидуально предоставленными
    аргументами;

apply()
  - вызывает функцию с указанным значением this и аргументами, предоставленными в
    виде массива.

Используйте .bind(), когда вы хотите, чтобы эта функция позже вызывалась с
определенным контекстом, полезным в событиях. Используйте .call()или .apply(),
когда вы хотите немедленно вызвать функцию и изменить контекст.

Call/apply вызывает функцию немедленно, тогда как bind возвращает функцию, которая
при последующем выполнении будет иметь правильный набор контекста для вызова
исходной функции. Таким образом вы можете поддерживать контекст в асинхронных
обратных вызовах и событиях.
