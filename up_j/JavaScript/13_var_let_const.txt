1) var (устаревшее объявление переменной)
  - область видимости переменных var ограничивается либо функцией, либо, если переменная глобальная, то скриптом.
    Такие переменные доступны за пределами блока, к примеру if или while, видны до объявления;

  - переменная var – одна на все итерации цикла и видна даже после цикла;

  - объявления переменных var обрабатываются в начале выполнения функции (или запуска скрипта, если
    переменная является глобальной).

2) let
  - видны только после объявления и только в текущем блоке - {...};

  - при объявлении переменной в цикле for(let …) – она видна только в этом цикле, каждой итерации
    соответствует своя переменная let.

3) const
  - объявление const задаёт константу, то есть переменную, которую нельзя менять;

  - в остальном объявление const полностью аналогично let.
