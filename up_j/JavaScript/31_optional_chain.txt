Опциональная цепочка '?.'
  - это безопасный способ доступа к свойствам вложенных объектов, даже если какое-либо
  из промежуточных свойств не существует.

  Опциональная цепочка ?. останавливает вычисление и возвращает undefined, если значение
  перед "?." равно undefined или null.

  Пример:

  Вариант без ?.
    let user = {}; // пользователь без свойства "address"

    alert(user.address.street); // Ошибка!
  
  Вариант с ?.
    let user = {}; // пользователь без адреса

    alert( user?.address?.street ); // undefined (без ошибки)
