Селектор (выборщик, переключатель) - позволяет выбрать с веб-документа необходимый элемент, определить,
к какому элементу применять то или иное CSS-правило.

Базовые селекторы:
  - селектор по элементу (тегу):
    div {
      padding-bottom: 15px;
    }

  - селектор по классу:
    .center {
      text-align: center;
    }

  - селектор по идентификатору (id):
    #footer {
      height: 50px;
    }

  - универсальный селектор - (селектор звёздочка) предназначен для выбора всех элементов:
    * {
      margin: 0;
    }

  - селектор по атрибуту - предназначен для выбора элементов по имени атрибута и (или) его значению:
    [target] {
      background-color: red;
    }
