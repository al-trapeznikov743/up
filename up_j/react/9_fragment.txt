Возврат нескольких элементов из компонента является распространённой
практикой в React. Фрагменты позволяют формировать список дочерних
элементов, не создавая лишних узлов в DOM.

return (
  <React.Fragment>
    <ChildA />
    <ChildB />
    <ChildC />
    ...
  </React.Fragment>
);

Сокращённая запись:
  <>
    <ChildA />
    <ChildB />
    <ChildC />
    ...
  </>
