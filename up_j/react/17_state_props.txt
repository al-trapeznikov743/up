- State это объект, который хранит динамические данные компонента и позволяет компоненту
  отслеживать изменения между рендерами.

- Props представляет коллекцию значений, которые ассоциированы с компонентом.
  Эти значения позволяют создавать динамические компоненты, которые не зависят от жестко
  закодированных статических данных.
