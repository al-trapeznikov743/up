Event - представляет собой любое событие, которое происходит в DOM;
  Некоторые из них генерируемые пользователем (клик мышью или нажатие клавиши на клавиатуре),
  а некоторые - генерируемые API (события, обозначающие завершение процесса анимации, приостановка видео и т.д.).

DevTools может показать прослушиватели событий JS, зарегестрированные в элементах.
DevTools -> Elements -> Event Listeners

DevTools также может привести к точной строке кода, где зарегестрировано событие.

Использование:
  - выбрав элемент на панели элементов можно просмотреть события, зарегестрированные на любом узле;
  - так же есть возможность тут же удалить событие и показать определение функции, определяющей событие.
