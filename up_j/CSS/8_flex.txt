Flexbox предоставляет инструменты для быстрого создания сложных, гибких макетов, и функции, которые были сложны в традиционных методах CSS.
flex - это сокращённое свойство, определяющее способность гибкого элемента растягиваться или сжиматься для заполнения собой доступного свободного пространства.

flex-direction - главная ось и направление
justify-content - выравнивание вдоль оси
align-items - выравнивание перпендикулярно оси
flex-basis - размер по основной оси
flex-grow - приорити растягивание
flex-shrink - приорити сжатие
