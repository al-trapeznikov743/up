Символ - это уникальный и неизменяемый тип данных, который может быть использован
как идентификатор для свойств объектов.

Создание символа:
  let id = Symbol();

При создании, символу можно дать описание (имя), в основном использующееся для
отладки кода:
  let id = Symbol("id");

Символы гарантированно уникальны. Даже если мы создадим множество символов с
одинаковым описанием, это всё равно будут разные символы. Описание – это просто
метка, которая ни на что не влияет.

Приведённый выше синтаксис, использующий функцию Symbol(), не создаст глобальный
символ, который был бы доступен в любом месте вашего кода. Для создания символов,
доступных во всех файлах и в окружении (глобальной области), используйте методы:
  - Symbol.for()
  - Symbol.keyFor()
  чтобы задать или получить символ из глобального символьного реестра.

Символы позволяют создавать «скрытые» свойства объектов, к которым нельзя нечаянно
обратиться и перезаписать их из других частей программы.

Метод Object.getOwnPropertySymbols() возвращает массив символов и позволяет
получить символьные свойства конкретного объекта.

Существует множество системных символов, используемых внутри JavaScript, доступных
как Symbol.*.
