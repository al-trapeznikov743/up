Yarn
  - относительно новый (2016) менеджер пакетов, разработанный facebook для среды
    выполнения JavaScript Node.js. Альтернатива npm.

  NPM в файле package.json фиксирует не конкретную версию используемых пакетов, а
  их диапазон. Таким образом, стратегия стандартного клиента npm может привести к
  тому, что на основе идентичных файлов package.json в разное время будут
  установлены различные версии пакетов.
  
  Yarn решает эту проблему, так как позволяет точно зафиксировать зависимости в
  файле yarn.lock.

  Ещё один плюс yarn перед npm это то, что последовательная установка пакетов через
  npm заметно замедляет работу. Yarn же поддерживает параллельную установку, что
  обычно в несколько раз быстрее.

  По последним данным, несмотря на то, что yarn имеет некоторые преимущества,
  скорости yarn и npm в их последних версиях довольно сопоставимы.
