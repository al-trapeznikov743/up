Медиа-запросы, медиавыражения
  Специализированные правила CSS, позволяющие управлять стилями, адаптировать их в
  зависимости от:
    - типа отображения устройств (принтер, десктоп, смартфон);
    - характеристик устройства (ширина окна просмотра браузера);
    - внешней среды (внешнее освещение).

  Синтаксис
    @media условие {
      стили, удовлетворяющие условию
    }
    
  Я использовал адаптив под ширину браузера, например:
    @media (max-width: 900px) {
      .grid {
        grid-template-columns: 2fr 1fr;
      }
    }
  
  Использовал вариант "при печати":
    @media print {
      header {
        background: none;
        color: #000;
      }
    }
